## how to use 

- cd fe ; npm install

- add password to api/app.js

- cd api ; node app.js 

    - runs on http://localhost:3000/{command}/{params} that hits http://pokeribarelyknowher.com/api/{command}{password}{params}{json}'

- ng serve --open - runs on http://localhost:4200
